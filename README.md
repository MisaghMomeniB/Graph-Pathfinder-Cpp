## ðŸš€ Usage Examples

### CLI Usage

Assuming a graph file `simple_graph.txt` with:

```
5 6
0 1 2.5
0 2 1.5
1 3 1.0
2 3 2.0
2 4 3.0
3 4 1.2
```

Run:

```bash
./graph_pathfinder --algo dijkstra --input simple_graph.txt --start 0 --end 4
```

Expected output:

```
Path found: 0 -> 2 -> 3 -> 4
Total cost: 4.7
```

### Library Integration

```cpp
#include "GraphPathfinder.hpp"

int main() {
    Graph g(5);
    g.addEdge(0,1,2.5);
    g.addEdge(0,2,1.5);
    g.addEdge(1,3,1.0);
    g.addEdge(2,3,2.0);
    g.addEdge(2,4,3.0);
    g.addEdge(3,4,1.2);

    Pathfinder pf(g);
    auto result = pf.aStar(0, 4, [](int u, int v){
        // Zero heuristicâ€”equivalent to Dijkstra
        return 0.0;
    });

    if (result.found) {
        std::cout << "Path: ";
        for (auto n : result.path) std::cout << n << " ";
        std::cout << "\nCost: " << result.cost << "\n";
    }
    return 0;
}
```
